# Otros

Esto es bien rápido. Esta es la referencia para cualquier check que se haga en el sistema, dependiendo de el stat requerido, funciona con esta tabla:
#### Tabla de Dificultad

| Dificultad   | Valor Objetivo (d20+Skill ≥) |
| ------------ | ---------------------------- |
| Trivial      | 5                            |
| Fácil        | 10                           |
| Moderada     | 15                           |
| Difícil      | 20                           |
| Muy Difícil  | 25                           |
| Extrema      | 30                           |
| Caso Perdido | 35+                          |
Eso sería. Bien simple. Al ser un check de múltiples skills (un check que requiera Convicción y Calma, por ejemplo), se le baja el escalón del d20 acorde a la cantidad de skills dentro del check. Por ejemplo, si son 2 skills, 1d12, si son 3, 1d8. Así.

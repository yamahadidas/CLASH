Acciones

Escalón de arma: 0 = ideal, 1 = "asequible", 2 = "ineficiente" (o el valor que defina tu arma). Se suma al escalón de dado dictado por el N° de acciones, sobre la escala 1d20 → 1d12 → 1d8 → 1d6 → 1d4 → 1d2.
Nota: el ejemplo de arma del documento original (Espada Inflamable, 3 acciones) da 1d8 / 1d6 / 1d2 para Corte/Perforación/Contusión. Siguiendo la escala de forma estrictamente aditiva, el último debería salir 1d4, no 1d2 — probable error de tipeo en el original. Esta calculadora usa la versión aditiva consistente; ajusta el escalón manualmente si tu mesa prefiere seguir el texto tal cual.

# Nivel de Emoción

![](emotion.png)

Una vez ocurran las siguientes ocurrencias en un turno, el nivel de **Emoción** de un personaje puede aumentar. Esta es una mecánica universal y no se influye por estadísticas.

Ejemplando:

* Si se gana un CLASH. (+1 Emoción)
* Si se inflige daño máximo o alrededor de este. (+2 Emoción)
* Si se inflige daño *mínimo* o alrededor de este. (+2 Emoción)
* Se aturde a un enemigo. (+2 Emoción)
* Se mata a un enemigo. (+3 Emoción)

Cuando se acumulen al menos 5 de estos, el personaje ganará un **Slot de ACCIÓN**, el cual le permitirá declarar un ATAQUE adicional por turno. Esto aplica para amigos y enemigos.

Hay un segundo limite de Emotion Level conseguido cuando se consiguen 8 puntos. Al entrar a este threshold, el personaje obtiene un ATAQUE Inmodificable, poderoso y considerado algo similar a una "ulti" del mismo. Este ATAQUE es definido por el GM.
